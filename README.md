# How to Use the AskOntology proof-of-concept with the MOAFDITO Ontology

This tool lets you ask natural language questions about the MOAF-DiT ontology. It retrieves relevant concepts from the schema and generates an explanation using AI. If you find something missing, you can propose a new concept directly from the interface.

Use the AskSchema for any ontology: https://z-arghavan.github.io/AskSchema2/
Use the AskSchema for MOAFDITO ontology (ontology is pre-loaded): https://z-arghavan.github.io/AskSchema2/?ontology=moafdito
MOAFDIT ontology: https://z-arghavan.github.io/MOAF-DiT/

---

## Step 1 - Get a Gemini API key

Go to https://aistudio.google.com/app/apikey and create a free API key. You will need this to get AI-generated explanations.
Your Gemini API key stays in your browser and is never shared.

---

## Step 2 - Enter your API key

Paste your Gemini API key into the API Key field and click Save.

---
## Step 3 - check if the ontology is loaded.

Open the link you received. The ontology is already loaded. You will see the setup screen with one field to fill in.

If the link does not have the ontology pre-loaded, you will also need to upload a Turtle (.ttl) file in Step 2 of the setup screen.


---

## Step 4 - Build the index

Click "Build index". This sends the ontology concepts to Gemini API to create a semantic index. It takes about 20 seconds.

The "String-match only" is a safety mechnaism in case the API key fails. The tool will still work but with lower accuracy.


---

## Step 5 - Ask questions

Type a question in plain English and click Ask, or press Ctrl+Enter.

Examples:
- Does a Defect class exist?
- What properties does WorkOrder have?
- Is there a concept for solder paste inspection?
- How to use the class Prdouct?

The tool will show concepts from the ontology that are related to your question, followed by an explanation generated by AI.

You can ask follow-up questions. There is bottom for it. If you do that, the tool remembers the context of the current conversation. To start fresh, click "New conversation".

---

## Step 6 - Propose a concept

After each answer, an "AI-generated suggestions" section appears. If you think a concept is missing or could be improved, fill in the form and click Submit proposal.

Fields:
- Concept name: the name you are proposing (required)
- Type: Class, Object property, or Data property
- Parent / related concept: an existing concept it connects to
- Description: what it represents and why it matters
- Example usage: a sentence showing how it would be used

Your proposal is recorded and sent to Arghavan for review and improvement of the ontology.

---

## Notes

- You can ask multiple questions in one session. All of them are logged in the Conversation log at the bottom of the page.
- Your API key is saved in your browser (we will not see it).
- All responses come from the ontology schema, not from a knowledge graph or external database.
